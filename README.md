# eSocial Management System

Application web développée en Java permettant de gérer les employeurs et les assurés dans un système de gestion sociale.

## Description

Cette application permet aux entreprises de gérer les employeurs et les employés assurés.  
Elle permet également d’associer les employés à leurs employeurs et de gérer les informations liées aux salaires.

## Fonctionnalités

- Gestion des employeurs
- Gestion des assurés
- Association des assurés aux employeurs
- Modification du salaire
- Consultation des employés par employeur

## Technologies utilisées

- Java
- Jakarta Servlet
- JSP / JSTL
- JPA / Hibernate
- MySQL
- Apache Tomcat
- HTML / CSS

## Architecture

Le projet suit l’architecture MVC :

- Model : Entités JPA
- View : Pages JSP
- Controller : Servlets
